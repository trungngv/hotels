Distribution

We can divide hotels into 3 categories based on their capacity: small (0 - 50 rooms), medium (50 - 100 rooms), large (over 100 rooms). There are a few hotels in all regions with very large capacity (over 300 rooms), these are exceptions and we ignore in this analysis.

It's clear from the distribution that most hotels in Hanoi and Nha Trang are small. Although in Nha Trang the market for medium to large hotels is bigger.

Hypothesis: land is less expensive in Nha Trang than HN; hotels in Nha Trang more spread out.

def convert_rating(x):
    if ' hotel' in x:
        return x.split()[0]
    elif 'stars' in x:
        return "%s-star" % x.split()[0]

def get_ratings(df):
    ratings = df.starRating.astype(np.string_)
    ratings = ratings[ratings != '']
    return ratings.map(convert_rating)

Percentage of star-ratings

def plot_review_scores(df, sp):
    scores = df.ix[df.avgReviewScore != '', 'avgReviewScore'].astype(np.float32)

from collections import Counter

def to_int(xs):
    return [int(x) for x in xs]

def get_nation_counts(df):
    c = Counter()
    for (i, names) in enumerate(df.topNationNames):
        counts = df.topNationCounts.values[i]
        counts = counts.replace('[', '').replace(']','')
        d = dict(zip(names.split(','), to_int(counts.split(','))))
        c.update(d)
    return c

def plot_nation_counts(df, top=20):
    counter = zip(*(get_nation_counts(df).most_common(top)))
    pos = np.arange(len(counter[0]), 0, -1)
    plt.barh(pos, counter[1], align='center')
    plt.yticks(pos, counter[0])

plt.figure()    
plot_nation_counts(df_hn, 20)
plt.figure()
plot_nation_counts(df_nt, 20)
plt.figure()
plot_nation_counts(df_pq, 20)
